Game Hub :
"The Hub For Gamers"


Game Hub is a video game digital distribution service and storefront by KodeGo. It was launched as a software client in November 2022 as a way for KodeGo to provide automatic updates for their games, and expanded to distributing and offering third-party game publishers' titles in late 2021.

Game Hub offers various features, like digital rights management (DRM), game server matchmaking, anti-cheat measures, social networking and game streaming services. It provides the user with automatic game updating, saved game cloud synchronization, and community features such as friends messaging, in-game chat and a community market.
